    "| Command | Description | \n",

    "| `pwd `| print working directory |\n",

    "| `rmdir [dir]` | remove an *empty* directory |  \n",
    "| `cp [file] [dir]` | copy a file to a directory |\n",

    "| `mv [file] [dir]` | move a file to a directory |\n",
    "| `mv [filename1] [filename2]` | rename a file |\n",
    "| `mv [dir1] [dir2]` | move a directory to another directory |\n",
    "| `rm [file]` | remove a file |\n",
    "| `rm -r [dir]` | remove a directory and all contents (**use with caution!**) |\n",
